Задача: 
Нужно разработать бота который будет учитывать 
накопленные бонусы при совершении покупок.

1. Сделать кнопку "Мои бонусы":
   отображает все накопленные бонусы по всем 
   поставщикам пользователя.

2. Сделать кнопку "Настроить бонусы", по 
   нажатию можно задать % от суммы 
   начислять в виде бонусов и какой % 
   начислять по реферальной программе, 
   какой процент от суммы покупки можно 
   максимально списать в качестве бонуса, 
                   !attention!
   !управлять изменениями в этом разделе может   !
   !только владелец почты на которую отправляется!
   !код подтверждения после ввода которого       !
   !можно производить изменения кода.            !

3. Сделать кнопку "Начислить бонусы" 
   по нажатию на нее можно ввести сумму 
   покупки, в ответ программа пришлет ссылку 
   по нажатию на которую получатель 1 раз 
   может получить бонусы на свой баланс.

4. Сделать кнопку "Списать бонусы" по нажатию 
   на нее нужно указать код который спрашиваем 
   у клиента и сумму покупки, 

5. бот ищет номер в базе и удаляет бонусы клиента 
   по данному поставщику, покупателю приходит 5
   значный код который оператор должен ввести 
   для завершения списания.

6. Сделать кнопку "Реферальная программа" по 
   нажатию приходит сообщение с описанием реферальной
   программы и ссылкой, кликнув на которую приглашенный
   закрепляется за рефералом и рефералу начинают
   начисляться проценты по всем покупкам.

7. Сделать кнопку "Получить бонусы" по нажатию приходит
   5 значный код подтверждения, который человек должен
   сказать оператору для начисления или списания бонусов

8. Сделать кнопку "О нас" по нажатию приходит информация для обратной связи с владельцем бота.

 ->После начисления и списания бонусв нужно отправлять уведомления
 ->После проведения покупок приглашенными по реферальной ссылке их рефералам
   нужно присылать уведомление в бот с количеством начисленных бонусов.
 ->Нужно предусмотреть возможность владельцу бота просматривать базу данных
   всех бонусов и пользователей зарегистрированных в системе.
 ->Каждую функцию в коде нужно сопроводить комментарием описывающим что она делает.
